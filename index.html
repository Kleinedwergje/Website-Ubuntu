Mappen en bestanden

  • Huidige map tonen: pwd
  • Inhoud van map bekijken: ls
  • Inhoud met details: ls -l
  • Inhoud inclusief verborgen bestanden: ls -a
  • Naar map navigeren: cd <mapnaam>
  • Één map omhoog: cd ..
  • Home directory: cd ~
  • Map maken: mkdir <mapnaam>
  • Bestand maken / openen: nano <bestandsnaam>
  • Bestand kopiëren: cp <bron> <doel>
  • Bestand verplaatsen/naam wijzigen: mv <bron> <doel>
  • Bestand verwijderen: rm <bestandsnaam>

Bestanden bekijken

  • Alles in één keer bekijken: cat <bestandsnaam>
  • Begin van bestand bekijken: head -n 10 <bestandsnaam>
  • Einde van bestand bekijken: tail -n 10 <bestandsnaam>
  • Scrollbaar bekijken: less <bestandsnaam>
  • Zoek in bestand: grep <zoekterm> <bestandsnaam>
  • Live volgen van log: tail -f <bestand>

Netwerk

  • Huidig IP-adres bekijken: ip a
  • Ping netwerkhost: ping <ip of host>
  • Netplan configuratie map: cd /etc/netplan
  • Netplan bestand openen: sudo nano 50-cloud-init.yaml
  • Wijzigingen toepassen: sudo netplan apply
  • IP configuratie testen: ip a

Logs

  • Logbestand scrollbaar bekijken: sudo less /var/log/syslog
  • Laatste 10 regels: sudo tail -n 10 /var/log/syslog
  • Eerste 10 regels: sudo head -n 10 /var/log/syslog
  • Hele log in één keer: sudo cat /var/log/syslog
  • Specifieke fouten zoeken: sudo grep error /var/log/syslog

GitHub

  • Repository klonen: git clone <url>
  • Status bekijken: git status
  • Wijzigingen toevoegen: git add <bestandsnaam>
  • Commit maken: git commit -m "bericht"
  • Pushen naar GitHub: git push
  • Branch maken: git branch <naam>
  • Branch wisselen: git checkout <naam>
